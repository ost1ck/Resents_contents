# Recents Content App

Flutter application that shows user's recent activity from platforms like YouTube, Spotify, and Megogo.

## Features
- Firebase Authentication
- Recent content from multiple platforms
- Feature-based architecture

## Tech Stack
- Flutter
- Dart
- Firebase

## Setup
1. Clone repo
2. Run flutter pub get
3. Run the app
